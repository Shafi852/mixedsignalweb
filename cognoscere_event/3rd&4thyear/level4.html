Question

Miss Mary plans to start a small dairy, but prior to this ,she needs to get the equipments ready. She decides to have a centrifugation system with the following specifications:

1. It has mode switch with 2 modes :
   a) Setting time: The total time duration for which the system operates is set.
   b) Setting RPM: The percent of the Rpm is set.( Say 70% of total speed)

2. Two switches D1 and D2:
This is used to set the time duration and the percent of the Rpm by the user. By pressing the switches D1 and D2, the count value of time and rpm can be set.

3. Start switch : Once the parameters are set, upon pressing the start button,the centrifugal system must revolve with the given speed for the given duration.

Now, it's your task to help Miss Mary at her small entrepreneurial venture.Hurry up and go get started.

Note: The user has to explicitly set the Time and rpm values ( As in an oven). Once the values are set ,press the start button. The motor has to rotate with that speed for the time duration which was set by the user.
